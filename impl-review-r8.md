Re-verified the current implementation against `SPEC.md` and all prior findings in `impl-review-r1.md` through `impl-review-r7.md`. Prior issues not listed below appear closed. `npm run lint` and `npm run build` both pass on the current tree.

[MAJOR] Section 12: Slow Gemini timeouts still surface as hard failures — `src/lib/gemini.ts` caps Vertex calls at 5 seconds and throws `LLM analysis timed out...`, and `src/app/api/analyze/route.ts` maps that path to `503 LLM_ERROR` instead of the spec's seamless timeout fallback. A slow AI call therefore still lands on the error state with "Use Demo Data" instead of returning `200 { success: true, fallbackReason: "timeout" }` with simulated payload data. — fix: propagate a distinct timeout sentinel out of the Gemini/analyze layers and convert that path into the normal simulated success response before the 8-second route deadline, reserving `LLM_ERROR` for non-timeout AI failures.

[MAJOR] Section 10: `PRIMARY_FORM_UNCERTAIN` still misses the single-low-score case — `src/lib/analyze-page.ts` only sets `shouldRaiseUncertain` when `formCandidates.length > 1`, so a lone weak candidate form with `score < 60` is still accepted even though the spec requires `422 PRIMARY_FORM_UNCERTAIN` whenever the top score is below 60. The deterministic primary-form heuristic from earlier rounds is therefore only partially implemented. — fix: make the `< 60` threshold independent of candidate count and keep the top-two-delta check as the 2+-form condition.

[MINOR] Section 14: Demo Mode badge still disappears for exempt-IP bypasses — `checkRateLimit()` marks requests as `bypassed` for `RATE_LIMIT_EXEMPT_IPS`, but `getRateLimitHeaders()` only emits `X-Demo-Mode: true` when `FORCE_DEMO_MODE` is enabled. Because `src/app/page.tsx` and `src/app/analyze/page.tsx` rely on that header/persisted bit, the required "Demo Mode" badge is still missing for the exempt-IP presentation path. — fix: emit the client-facing bypass flag for any bypassed request, or expose a dedicated header/state that covers both force-demo and exempt-IP bypasses.
